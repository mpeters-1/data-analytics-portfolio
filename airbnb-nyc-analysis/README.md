# Airbnb NYC Data Analysis (Python + Pandas)

## Project Overview

This project analyzes Airbnb listings in New York City using Python. The goal is to explore host behavior, pricing patterns, neighborhood trends, and estimate revenue performance using exploratory data analysis (EDA) techniques.

The project focuses on building practical data analysis skills using **Pandas** and **Matplotlib**, including filtering, grouping, aggregation, and visualization.

---

## Tools & Libraries

- Python
- Pandas
- Matplotlib

---

## Dataset

The dataset contains Airbnb listings in NYC with features such as:

- Host information (host_id, host_name)
- Location (borough, neighborhood)
- Room type
- Price
- Reviews and availability metrics

---
