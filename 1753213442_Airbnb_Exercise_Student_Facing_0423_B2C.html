    Appendix: Python Cheat Sheet¶

    SELECTING COLUMNS

    - dataframe['a'] -- Select column a from a dataframe individually
- dataframe[['a','b']] -- Select columns a and b from a dataframe
- dataframe[:5] -- Select the first five rows from the dataframe
- dataframe[2:] -- Select all the rows of the dataframe starting from row 2 onwards
- dataframe.head(n) -- Select the number of rows in the dataframe specified by n
- dataframe.loc[row,column] -- Selects the number of rows, and columns specified with labels
- dataframe.iloc[row,column_index] -- Select the number of rows, and columns referencing their index

    LIBRARIES

    - import pandas as pd -- This imports the pandas library that contains the methods you need to manipulate tabular data
- import numpy as np -- This imports the numpy library that contains the np.where (IF) method you need to utilize
- import warnings -- This is the warning library that can be imported to either highlight or hide warning messages
- warnings. filterwarnings('ignore')--This ignores Python warning errors
- import matplotlib.pyplot as plt -- This is the visualization library needed to create plots

    OPERATORS

    - < - Less than
- > - Greater than
- <= - Less than or equal
- >= - Greater than or equal
- <> - Not equal
- = - Equal
- & - AND (e.g. dataframe['Column']>=1) & (dataframe['Column']<=2)
- | - OR

    DESCRIPTIVE STATISTICS FUNCTIONS

    - dataframe['column'].mean() - Returns the average value of a column
- dataframe['column'].min() - Returns the minimum value of a column
- dataframe['column'].max() - Returns the minimum value of a column
- dataframe['column'].sum() - Returns the sum of a column
- dataframe['column'].median() - Returns the median of a column

    SUBSETTING COLUMNS

    - dataframe['column'].isin(['A','B']) - Returns a T/F boolean list of entries matching A and B which are textual fields
- dataframe[condition] - Provided condition is a T/F boolean expression, it will return all matching entries that are
True

    USING NP.WHERE() AS A PROXY FOR IF STATEMENTS

    - np.where(dataframe['column']> 1, "Return when True", "Return when False") -- A True/False expression comparison for numerical terms.
- np.where(dataframe['column'].isin(['A','B']),"Return when True", "Return when False") -- A True/False expression comparison for textual comparisons.

    SORTING VALUES

    - dataframe.groupby(['A']).sum()[['B','C']].sort_values(by=['B'], ascending=False) -- The syntax to sort a dataframe using specific columns
- dataframe.groupby(['A']).sum()['B'].sort_values(ascending=False) -- The syntax to sort a dataframe provided you only have one numerical column to sort on.

    DICTIONARIES AND AGGREGATIONS

    - Dictionaries are special data types in Python defined by a key-value relationship.
Syntax wise, they use the {'key': 'value'} syntax
- .agg(['X','Y']) represents a advanced method of providing multiple descriptive statistics where X and Y represents aggregation functions (i.e. sum, mean, min, max etc.)
